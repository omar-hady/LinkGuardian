Command Line Interface

Use the CLI to analyze URLs from your terminal. The CLI mirrors the web heuristics and thresholds.

# Single URL
python app/cli.py https://example.com

# Batch (one URL per line)
python app/cli.py --batch test_urls.txt

From the project root you can also use the runner:

python run.py predict https://example.com
python run.py batch test_urls.txt
