About

LinkGuardian helps you assess the phishing risk of URLs using a transparent set of heuristics. We analyze only the URL string and public metadata; we do not execute scripts or run third-party trackers.

Privacy

Submitted URLs are processed in-memory and a summary of each analysis (URL, decision, score, timestamp) is stored locally to power the Stats page. Clear the cache or data anytime from Stats.

Limits

To keep the service responsive, we apply a small per-IP rate limit. Heavy users should prefer the CLI for batch runs.
